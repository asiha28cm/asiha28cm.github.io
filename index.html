Web Designer Sho Tanaka

東京都在住のWebデザイナー、田中翔です。会話でのコミュニケーション分野の会社を経営していますが、インターネットを通じたコミュニケーション分野への事業拡大のため、オンラインスクールでWeb制作について学んでいます。基本的なことから最新の表現方法まで実装できる知識と技術を身につけ、現代社会におけるコミュニケーションの専門家を目指しています。

Skills

Web

テーマを活用したWordPressサイトを制作できます。HTML/CSSは現在勉強中で、JavaScriptやjQuery、SEO対策にも挑戦する予定です。

Talk

会話でのコミュニケーションをコンサルティングしています。男女のコミュニケーションの違いや恋愛について、ビジネスシーンにおける会話や印象について等、様々なシーンにおけるコミュニケーション術を学べる場を提供しています。

self enhancement

自分磨きについてコンサルティングしています。モチベーションアップや第一印象の改善を目的とし、人の輪の中で輝ける自分作りができるようになるまでサポートしています。
